# Data Analyzer CLI

Simple Python CLI app to analyze numbers.

## Features:
- Add numbers
- Show data
- Calculate:
  - min, max, sum, average
  - even/odd count
  - greater/smaller than X
- Sort data (ascending/descending)

## How to run:
python data-analyzer-cli.py

## Example usage:
1. Run the script
2. Choose option from menu
3. Add numbers and analyze them
